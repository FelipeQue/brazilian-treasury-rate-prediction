# TreasuryCaster: Modelo Preditivo para Taxas de Leilões do Tesouro Nacional
Modelo preditivo para taxas de leilões do Tesouro Nacional utilizando regressão e variáveis macroeconômicas. Projeto desenvolvido para o curso Desenvolvimento de IA para Análise Preditiva do programa SCTEC.

## Os Dados

- Dataset de taxas de leilões do Tesouro Nacional. (Kaggle)
- Taxas de juros Selic. (API do Banco Central do Brasil)
- Taxas de câmbio do dólar. (API do Banco Central do Brasil)

## 🛠️ Tecnologias e bibliotecas utilizadas
- Python 3.14.3
- Jupyter Notebook 7.5.7
- Pandas 3.0.3
- Numpy 2.5.1
- Matplotlib 3.11.0
- Seaborn 0.13.2
- Scikit-Learn: 1.9.0
- Statsmodels: 0.14.6

- Durante o desenvolvimento deste projeto foi utilizado um ambiente virtual (venv) para gerenciar as dependências do projeto, garantindo que as bibliotecas necessárias estejam isoladas.
